# Setup
  + required: PHP >= 8.1
  + Used: Laravel 10
  + Step by step:
      1. clone source
      2. cd ecommerce/api
      3.  composer install
      4.  php artisan migrate
      5.  php artisan db:seed
      6.  php artisan serve
